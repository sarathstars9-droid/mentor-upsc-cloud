# UPSC Mentor OS — Nodemappers Starter

This starter gives you a future-proof `backend/nodemappers` layer with:
- central mapping entrypoint
- subject registry
- core normalization/tokenization/scoring/confidence logic
- starter subject mappers
- phase resolvers for current and future phases
- sample `findMicroTheme.js` integration file

## What to copy into your project
Copy `backend/nodemappers/` into your project.

## Immediate integration
Replace your current `findMicroTheme.js` body with the sample in:
- `samples/findMicroTheme.integration.example.js`

## Do you need to edit syllabus files now?
No, not immediately.
Only update syllabus files when:
- a node is actually missing
- a node code is wrong
- a truly essential alias/keyword is absent

Do NOT add analytics or planner logic into syllabus files.
